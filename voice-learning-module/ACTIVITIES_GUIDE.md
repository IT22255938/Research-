## ðŸ”¢ 1. Counting Adventure

**Character**: Sophie  
**Story**: Help Sophie count apples on her adventure through the apple orchard!  
**Target Age**: 4-8 years  
**Duration**: 10 minutes  
**Difficulty Range**: 1-5 (Adaptive)

### Learning Objectives
- Recognize numbers by quantity
- Count objects accurately
- Understand one-to-one correspondence
- Build number naming skills

### Question Types
- **Difficulty 1**: Count 1-3 objects (apples, stars, circles)
- **Difficulty 2**: Count 4-6 objects (hearts, triangles, flowers)
- **Difficulty 3**: Count 7-8 objects (butterflies, apples)
- **Difficulty 4**: Count 9-10 objects (balloons, stars)
- **Difficulty 5**: Mixed counting scenarios with story context

### Story Elements
- Setting: Beautiful sunny apple orchard with friendly trees
- Character: Sophie (needs help counting apples)
- Ambient sounds: Orchard sounds, celebration sounds
- Educational tone: Encouraging and supportive

### Adaptive Engine
- **Increase difficulty**: If student achieves 75%+ accuracy
- **Decrease difficulty**: If accuracy drops below 50%
- **Minimum difficulty**: 1 (very easy)
- **Maximum difficulty**: 5 (very hard)

### Rewards
- **XP per correct answer**: 10 XP
- **Bonus for speed**: 5 XP (answered in <5 seconds)
- **Bonus for accuracy**: 10 XP (if 100% accuracy on all questions)
- **Badges earned**: "Counter Expert" ðŸ†

### Example Questions
```javascript
// Difficulty 1 - Very Easy
Q: "Sophie found one apple. How many apples does she have?"
A: "one" / "1" / "one apple"

// Difficulty 3 - Medium
Q: "Sophie picked 7 apples. How many apples did she pick?"
A: "seven" / "7" / "seven apples"

// Difficulty 5 - Hard
Q: "Sophie counted apples in 3 baskets: 2, 3, and 2 apples. How many total?"
A: "seven" / "7" / "seven apples total"
```

## ðŸŽ¯ Gamification Integration

All activities are integrated with the gamification system:

### XP Rewards
- **Counting Adventure**: 10 XP per correct answer
- **Number Recognition**: 10 XP per correct answer
- **Math Mystery Solver**: 15 XP per correct answer (higher value)
- **Alphabet Adventure**: 10 XP per correct answer

### Speed Bonuses
- Answer within time limit (12-15 seconds): +3-5 XP bonus
- Encourage quick thinking and confidence

### Accuracy Bonuses
- Achieve 100% accuracy on all activity questions: +10-20 XP bonus
- Rewards thorough understanding

### Level Progression
- **Formula**: Level = floor(Total XP / 100) + 1
- **Level 1**: 0-99 XP
- **Level 2**: 100-199 XP
- **Level 3**: 200-299 XP
- **And so on...**

### Badge Awards
- Each activity has associated badges
- Badges awarded on first completion
- Display on student gamification dashboard
- Track achievement progress

### Leaderboard Integration
- XP accumulation across all activities
- Real-time leaderboard updates
- Top 3 learners displayed with medals
- Rank progression visible to students

## ðŸ“Š Adaptive Difficulty Algorithm

All activities use the same adaptive difficulty system:

```
IF student_accuracy >= 0.75:
  difficulty += 1 (up to max)
ELSE IF student_accuracy < 0.50:
  difficulty -= 1 (down to min)
ELSE:
  difficulty = unchanged
```

**Purpose**: Keep students in "optimal challenge zone"
- Not too easy (boredom)
- Not too hard (frustration)
- Scaffolded learning progression

**Personalization**: Each student has independent difficulty level per activity
- Emma: Counting Adventure at Level 3
- Lucas: Number Recognition at Level 1
- Sophia: Math Mystery Solver at Level 2

## ðŸ“ž Support & Troubleshooting

**Activity not loading?**
- Check that activity files exist in `src/activities/`
- Verify activity is exported in `index.js`
- Check browser console for errors

**Voice input not working?**
- Ensure microphone permissions granted
- Check browser supports Web Speech API
- Try refreshing the page

**Difficulty not adapting?**
- Verify adaptive rules in activity config
- Check accuracy calculation
- Review student response recording

**Badges not awarding?**
- Check badge conditions in activity
- Verify database is storing responses
- Review backend badge API
